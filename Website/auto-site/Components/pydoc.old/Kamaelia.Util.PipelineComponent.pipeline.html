Kamaelia.Util.PipelineComponent.pipeline

class pipeline(Axon.Component.component)

pipeline(*components) -> new pipeline component.

Encapsulates the specified set of components and wires them up in a chain (a pipeline) in the order you provided them.

Arguments: - components - the components you want, in the order you want them wired up

Inboxes

  • control : Secondary inbox often used for signals. The closest analogy is unix signals
  • inbox : Default inbox for bulk data. Used in a pipeline much like stdin

Outboxes

  • outbox : Default data out outbox, used in a pipeline much like stdout
  • signal : The default signal based outbox - kinda like stderr, but more for sending singal type signals
