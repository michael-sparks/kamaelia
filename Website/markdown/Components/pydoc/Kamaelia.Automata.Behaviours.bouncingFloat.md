-----------------------------------------------------------------------------------------------------

bouncingFloat(scale\_speed) -\> new bouncingFloat component

A component that emits a value that constantly bounces between 0 and 1.

scale\_speed scales the rate at which the value changes. 1.0 = tenths,
0.5 = twentieths, etc.
